# Truth Field Detector 🛰️

> Real-time narrative risk assistant that detects emotional manipulation and misinformation patterns using Amazon Nova.

## What it does
Truth Field Detector analyzes news, video, and social media content in real time. It detects emotional language, absolutist claims, missing citations, and narrative inconsistencies, delivering a Narrative Risk Index to the user.

## Built with
- Amazon Nova 2 Lite
- Amazon Nova 2 Sonic
- Amazon Nova Multimodal Embeddings
- Nova Act
- Python
- AWS Bedrock
